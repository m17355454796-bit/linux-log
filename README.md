# Linux Study Notes

这个仓库包含我的Linux学习笔记和相关代码练习。

## 目录结构

- `lesson/`: 课程相关代码和练习
  - `makefile/`: Makefile学习
  - `process/`: 进程相关代码
  - `process-study/`: 进程学习代码
- `linux-log/`: 学习日志，按日期记录
  - 包含从2026-04-04到2026-04-12的学习笔记
  - 涵盖操作系统基础、进程管理、虚拟地址空间等主题

## 学习内容

- 操作系统基础知识
- 进程创建、调度、退出
- 虚拟地址空间
- Makefile编写
- Linux系统编程

## 使用说明

1. 克隆仓库：
   ```bash
   git clone https://github.com/m17355454796-bit/linux-log.git
   ```

2. 查看日志：进入 `linux-log/` 目录阅读Markdown文件
3. 运行代码：进入 `lesson/` 子目录编译和运行相关程序
